Porque Aprender HTML5 y CSS 3

LinenBen17 - 28 de Diciembre del 2020

Hola!!!, Bienvenido seas A nuestro primer "post" del blog.
Si eres Nuevo en esto de la programación y no sabes Que Aprender Primero, Pues en este Post te lo explicamos Detalladamente.

Empecemos Definiendo HTML5 y CSS

HTML viene de las Siglas HyperText Markup Language (Lenguaje de marcado de hipertexto) el número 5 viene de la versión, Es la quinta version importante de HTML, Se complementa con las Hoja de estilo en cascada llamadas CSS, Que viene de las Siglas Cascading Style Sheets Este Se encarga de Darle los estilos al código HTML


Listo! ahora ya sabes la definición de HTML y CSS. Pero... ¿Porque aprenderlo?

1. Puedes Crear Tu mismo una Pagina Web


Ya sea, Para uso personal o tu negocio, Tener Una Pagina Web hoy dia indica la formalidad o Profesionalidad que hay en un Negocio o Empresa, también podrías trabajar para otras personas y hacer ganancias con ello.

2. Entretenimiento


El Proceso de crear una Pagina Web es un Proceso largo si no se sabe mucho, o no se tiene un Concepto (Sketch) de que realizar, Pero de igual Manera te puedes Entretener Jugando Con las Hojas de Estilos, Los códigos, y si piensas implementar JavaScript, Con los Errores al Programar, Aunque no lo Parezca es un Proceso Muy entretenido, Y si utilizas Editores de Texto como Vscode Podrías Instalar extensiones que te Ayudaran más al momento de Desarrollar o Programar.

3. Abre Puertas A Cosas Más Avanzadas


HTML y CSS es un Buen Comienzo Para Adentrarte en el Mundo de la Programación, ¿Como..?
Pues te ayuda a crear algo llamado Lógica de Programación, Habrá situaciones en las que obligatoria mente debas utilizar lenguajes de Programación Como JavaScript, PHP, Jquery, entre otras...
Y eso Te puede impulsar a aprender otras cosas y aprender más cada día.

Listo Ahora Ya sabes que puedes hacer con HTML5 y CSS3, el único Que Puede Dar la iniciativa de Aprender Eres Tú y Recuerda el Único Limite Que hay es la Mente
